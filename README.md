# Sanjha Virsa

A free multilingual educational platform for Punjabi language, history, culture, and heritage.

## Mission

Sanjha Virsa means shared heritage. The goal is to help Punjabi children growing up outside Punjab learn language, culture, identity, and history in an interactive digital format.

## Current Status

This repository now has a working Next.js foundation that can deploy on Vercel.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel

## Run locally

```bash
npm install
npm run dev
```
